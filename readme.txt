== Description ==
A WordPress-WooCommerce plugin for merchants to integrate Tingg express checkout on their online shops, offering their customers a pan-african variety of payment options.

== Installation ==
Installation on the WordPress plugin store
1. Login into your WordPress site admin dashboard
2. Go to the plugins page
3. At the top of the page besides the `Plugin` title, click on the `Add New` button
4. Search for the plugin using the term `cellulant-tingg-integration-plugin` and the tags tingg, africa, payments, integration
5. Once found, return to the plugins page and activate the plugin

Manual installation
1. Download the plugin from https://tingg.africa/services/#developers
2. Go to your WordPress installation directory / folder and navigate to `wp-content/plugins/`
3. Extract the contents there
4. Go to the plugins page and activate the plugin

== Frequently Asked Questions ==
